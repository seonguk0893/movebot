## 페이지 및 기능 설명

### HOME 화면
  - 음성인식을 위한 마이크 버튼과 해당 과일 버튼
  - 마이크 버튼을 클릭하면 음성인식 시작(과일명 인식)
  - 좌측 상단 버튼을 클릭하면 무브봇 안내음성 추출
